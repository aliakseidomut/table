## Technologies
- React
- Ant Design
  
## Functionality
- **Display table** with the following columns:
  - `Name`
  - `Date`
  - `Number`
  - `Actions`
- **Add new row** to the table
- **Edit existing row**
- **Delete row**
- **Sort functionality** on all columns
- **Search bar** above the table
  
## Script for launching
```
npm run start
```
